# 🎵 VRRadio for FiveM

Replace GTA V's default radio stations with your own custom internet radio streams.

VRRadio is a lightweight FiveM resource that lets server owners integrate custom radio stations into their server. It is written entirely in JavaScript, is easy to configure, and uses FiveM resource metadata for simple setup.

## ✨ Features

* 🎧 Replace default GTA V radio stations
* 🌐 Stream your own online radio station
* ⚡ Lightweight and optimized
* 🛠️ Simple configuration through resource metadata
* 📦 Easy installation
* 🔄 Supports live internet radio streams
* 🎵 Perfect for roleplay communities, public servers, and private servers

## 📥 Installation

1. Download the latest release.
2. Extract the `VRRadio` folder into your server's `resources` directory.
3. Add the following to your `server.cfg`:

```cfg
ensure VRRadio
```

4. Configure your radio stations using the resource configuration.
5. Restart your server.
